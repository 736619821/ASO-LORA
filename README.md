# ASO-LORA

A Soft Orthogonal Low-Rank Adaptive Continual Learning Scheme Based on Attribution Scores

This repo releases our implementation for the T5 model.

## Setup

You can install the required libraries by running 

```
pip install -r requirements.txt
```

## Training and Evaluation

You can reproduce our experiments by simply running

```
bash scripts/t5_order1.sh
```
